Printtank

Syntax:

PRNTVEH.EXE [files]

Printtank generates text files containing the important parameters of the vehicle types listed in tabular form. The different files contain the same data but are sorted by different parameters.

As optional parameter you can also use a wildcard, default is *.VEH

Examples:

PRNTTNK evaluates all vehicle types
PRNTTNK A*.TNK evaluates only airplanes
PRNTTNK S*.TNK evaluates only ships
PRNTTNK G*.TNK evaluates only ground vehicles

The textfiles that are being generated are:

  • tanks.txt - sorted by filename
  • tanks0.txt - sorted by jamming
  • tanks1.txt - sorted by view
  • tanks2.txt - sorted by armor
  • tanks3.txt - sorted by speed
  • tanks4.txt - sorted by production cost energy
  • tanks5.txt - sorted by production cost material
  • tanks6.txt - sorted by fuel consumption
  • tanks7.txt - sorted by fuel tank
  • tanks8.txt - sorted by maximum weapon strength
  • tanks9.txt - sorted by weight
  • tanksa.txt - sorted by distance

The columns are titled with:

  • description
  • armor
  • production cost energy
  • production cost material
  • id
  • fuel
  • the eight height levels the unit can (or can not) move on
  • the eight height levels the unit can attack, the strength of the strongest weapon is given
  • distance ( fuel / consumption )
  • fuel consumption
  • ascent
  • view
  • jamming
  • weight
  • wait
  • load
  • max weight for a single loadable unit
