Contact Information

Mailing lists

We have set up a number of mailing lists:

asc-hq-announce
New versions of ASC, unit sets and other major additions to ASC will be announced here.

To subscribe, write a mail to asc-hq-announce-request@lists.sourceforge.net with subscribe in the subject.
Or use the web interface

asc_general
Everybody who wants to participate in email games, discuss new maps, features, the project in general or anything else should join the list.

To subscribe to ASC_general, write a mail to ASC_general-subscribe@yahoogroups.com.
Or use the web interface.

asc_development
This list is intended for discussions about programming. I imagine that the stuff here is quite uninteresting to non-programmers :-)

To subscribe to ASC_development, write a mail to ASC_development-subscribe@yahoogroups.com.
Or use the web interface.

asc-hq-german
This list is for general discussion about ASC, but in the German language. Although English is the primary language of ASC, Battle Isle has a large fan community in Germany and so has ASC.
Important discussions here will be summarized in the english mailing list.

To subscribe, write a mail to asc-hq-german-request@lists.sourceforge.net with subscribe in the subject.
Or use the web interface.

Direct Email

We have deliberately removed all personal email adresses from this site. ASC is not a one-person project (although it had been once...), and we want to avoid discussions by private mail which should better be done through the mailing list.

Bug reports should be send to
bugs@asc-hq.org
