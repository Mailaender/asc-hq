Demount

Syntax:

DEMOUNT containerfilename

DEMOUNT extracts all files from the specified container file. Existing files will be overwritten without any query!!! An error handling does not take place!

But it works only in the current directory. If you use it in an empty directory it can't do much harm. The program has otherwise to be enjoyed with caution ...

MOUNT and DEMOUNT are the only ASC tools that do not require any datafiles themself and who do not evaluate any search paths defined in the ASC configuration file.
